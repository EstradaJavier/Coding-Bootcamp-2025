Coding Bootcamp Curriculum - April 2025

Welcome to your 10-week journey with Grok (xAI)! Learn JavaScript, Python, Java, HTML, and CSS in IntelliJ IDEA. Daily sessions: 3 hours (9:00 AM - 12:00 PM PDT), April 8 - ~April 25, 2025. Save as bootcamp.html in C:\Coding\Bootcamp\curriculum.

Human Factors & Design

  • Pacing: 3-hour sessions, 5-10 min breaks hourly.
  • Visuals: Dark mode, Bootstrap, icons.
  • Progression: Beginner to intermediate, job-ready.
  • Tests: 80/100 pass score for lessons, weeks, and final.

Folder Structure

Steps:

  1. Open File Explorer (Windows key + E).
  2. Navigate to C:\ (double-click “This PC” > “C:”).
  3. Right-click empty space > New > Folder, name it “Coding”.
  4. Double-click “Coding” to open it.
  5. Right-click > New > Folder, name it “Bootcamp”.
  6. Inside “Bootcamp”, right-click > New > Folder, name it “curriculum”.
  7. Save this file as bootcamp.html in C:\Coding\Bootcamp\curriculum.

Create Project in Existing IntelliJ

  1. Open IntelliJ: Double-click desktop icon or search “IntelliJ IDEA” in Start menu.
  2. If a project is open: File > Close Project to return to Welcome screen.
  3. Click “New Project” on Welcome screen.
  4. In “New Project” window, select “Empty Project” (or “Java” if “Empty Project” isn’t listed).
  5. Set details:
    • Name: Type “Bootcamp”.
    • Location: Click folder icon, navigate to C:\Coding, create “Bootcamp” folder if needed (right-click > New > Folder), select C:\Coding\Bootcamp, click “Open”.
  6. Click “Create”.
  7. In Project pane (press Alt + 1 if hidden), right-click “Bootcamp” > New > Directory, name it “curriculum”.
  8. Repeat for “week1” to “week10”.
  9. Right-click “curriculum” > New > File, name it “bootcamp.html”.

Verify Python

  1. File > Settings (Ctrl + Alt + S).
  2. Click “Plugins” > “Installed” tab.
  3. Search “Python”, ensure “Enabled”. If not, check box, click “Apply” > “OK”, restart.

Add Live Edit

  1. File > Settings > Plugins.
  2. “Marketplace” tab, search “Live Edit”.
  3. Click “Install” > “OK”, restart IntelliJ.

Test Setup

  1. Double-click “bootcamp.html” in “curriculum”.
  2. Paste: 
    Test
  3. Right-click file > Open in Browser > Chrome. If “Test” shows, setup works.

Install & Configure Git

  1. Download Git: git-scm.com.
  2. Double-click installer, click “Next” through defaults, then “Finish”.
  3. In IntelliJ, press Alt + F12 (terminal).
  4. Type git --version, press Enter (e.g., “git version 2.43.0” confirms install).
  5. Configure: 
    git config --global user.name "Javier P Estrada"
git config --global user.email "estrada.javier.525@gmail.com"

Initialize Repo

  1. Open C:\Coding\Bootcamp in IntelliJ (File > Open, select folder, click “OK”).
  2. VCS > Enable Version Control Integration.
  3. Select “Git” > “OK”.
  4. On GitHub: Sign in, click “+” > “New repository”.
  5. Name it “Coding-Bootcamp-2025”, set public, click “Create repository”.
  6. Copy HTTPS URL (https://github.com/EstradaJavier/Coding-Bootcamp-2025.git).
  7. In IntelliJ: VCS > Git > Remotes > Add, paste URL, name it “origin”, click “OK”.
  8. Right-click “Bootcamp” in Project pane > Git > Add.
  9. VCS > Commit (Ctrl + K), type “Initial commit”, click green checkmark.
  10. VCS > Git > Push (Ctrl + Shift + K), click “Push”.

Strategies

  • Subfolders: Commit week1, etc.
  • Commits: After tasks.
  • Branches: For intermediate challenges (VCS > Git > Branches > New Branch).
